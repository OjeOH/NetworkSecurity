## About This Repository

This repository is a curated collection of hands-on networking and cybersecurity labs completed across two semesters of study. It represents a progressive journey — starting from the fundamentals of building and securing a network, all the way through enterprise-grade firewall configuration, VPN deployment, penetration testing, and intrusion detection.

Each project was implemented using industry-standard tools: **Cisco Packet Tracer**, **Palo Alto NGFW (PAN-OS 10.x)**, **Wireshark**, **SNORT**, **CentOS Linux**, and **VMware vSphere**.

> The goal of this portfolio is not to show what I studied — it's to show what I can **build**, **secure**, and **defend**.
